Frequently asked questions

Where do I get an OpenAI API key? Visit platform.openai.com/api-keys, sign up for a free account, and create a new key. Paste it into Spit → Settings → API Key.
The text isn't being typed — it goes to the clipboard instead Spit needs Accessibility permission to type automatically. Go to System Settings → Privacy & Security → Accessibility and make sure Spit is enabled. After granting permission, restart Spit.
How do I change the keyboard shortcut? Open Spit Settings (click the menu bar icon → Settings), go to the General tab, and click "Change" next to "Dictation shortcut". Press your desired key combination.
Can I use push-to-talk instead of toggle? Yes! In Settings → General → Keyboard Shortcut, enable "Push-to-talk" and assign a key. Hold it to record, release to transcribe.
Spit keeps transcribing in the wrong language Set the language explicitly in Settings → General → Language. "Auto-detect" uses Whisper's automatic detection — it's very accurate but may occasionally pick the wrong language for short utterances.
How do I restore my Pro purchase on a new Mac? Open Spit and it will automatically restore your purchase when you're signed into the same Apple ID. If it doesn't, go to Settings and click "Restore Purchases".
